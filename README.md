- 3 Product Bundles with savings

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm 9+

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit http://localhost:3000

### Build

```bash
npm run build
```

### Deploy to Netlify

```bash
npm run deploy
```

## 📋 Configuration

Copy `.env.example` to `.env` and update:

```
VITE_STOREFRONT_API_TOKEN=your-token
VITE_PUBLIC_STOREFRONT_API_TOKEN=your-public-token
STORE_DOMAIN=thejadeandbloom.com
SESSION_SECRET=your-secret-key
```
